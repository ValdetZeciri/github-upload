= Projektantrag FutBet

== Beschreibung

FutBet ist eine Fußball-Webapplikation, in der man gegeneinander, innerhalb der Top Ligen bzw. Wettbewerbe, wetten kann.


== Nutzen

* Spielspaß
* Fußballkentnisse unter Beweis stellen
